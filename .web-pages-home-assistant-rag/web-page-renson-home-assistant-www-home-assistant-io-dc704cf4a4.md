The Renson integration Integrations connect and integrate Home Assistant with your devices, services, and more. [Learn more] pulls in data from the Renson Endura delta device. Most of the sensors that can be monitored from inside the Android/iOS application can be monitored with this integration.

Manual configuration steps
If the above My button doesn’t work, you can also perform the following steps
manually:
Browse to your Home Assistant instance.
Go to Settings > Devices & services .
In the bottom right corner, select the
Add Integration button.
From the list, select Renson .
Follow the instructions on screen to complete the setup.
The Renson integration Integrations connect and integrate Home Assistant with your devices, services, and more. [Learn more] provides the following actions. Each link below opens a dedicated page with examples, parameters, and a step-by-step UI walkthrough.
Set Breeze ( renson.set_breeze )
Configures the Breeze function of the Renson ventilation unit.
Set pollution settings ( renson.set_pollution_settings )
Configures the pollution settings of the Renson ventilation unit.
Set timer ( renson.set_timer_level )
Runs the Renson ventilation unit at a chosen level for a set time.
For an overview of every action across all integrations, see the actions reference .
