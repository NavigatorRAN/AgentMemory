The Sanix integration Integrations connect and integrate Home Assistant with your devices, services, and more. [Learn more] allows you to get water/sewage level sensor measurements from the Sanix devices made by BIT Complex .
To set up the integration Integrations connect and integrate Home Assistant with your devices, services, and more. [Learn more] , go to the Sanix dashboard and sign in to your account.
Not anyone can pull your data from the API. You need to authenticate by using an API token. To retrieve your API token, on the Sanix dashboard , go to the Help page, and under System version , copy your API token. You will need this during setup in Home Assistant.

If the above My button doesn’t work, you can also perform the following steps
manually:

Go to Settings > Devices & services .
In the bottom right corner, select the
Add Integration button.
From the list, select Sanix .
Follow the instructions on screen to complete the setup.
