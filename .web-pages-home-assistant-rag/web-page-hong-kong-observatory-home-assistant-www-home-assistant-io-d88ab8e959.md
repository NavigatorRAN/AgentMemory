The Hong Kong Observatory integration Integrations connect and integrate Home Assistant with your devices, services, and more. [Learn more] retrieves weather data from the Hong Kong Observatory Open Data API for meteorological data in HKSAR.
There is currently support for the following platforms within Home Assistant:
Weather
It displays the weather of your selected location and the 5-day weather forecast for Hong Kong.

Manual configuration steps
If the above My button doesn’t work, you can also perform the following steps
manually:
Browse to your Home Assistant instance.
Go to Settings > Devices & services .
In the bottom right corner, select the
Add Integration button.
From the list, select Hong Kong Observatory .
Follow the instructions on screen to complete the setup.
