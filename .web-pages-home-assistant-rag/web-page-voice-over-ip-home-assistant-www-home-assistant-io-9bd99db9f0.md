Configuration
The VoIP integration Integrations connect and integrate Home Assistant with your devices, services, and more. [Learn more] enables users to talk to Assist using an analog phone and a VoIP adapter. Currently, the system works with the Grandstream HT801 . See the tutorial for detailed instructions.
As an alternative, the Grandstream HT802 can be used, which is basically the same as the previously mentioned HT801, but has two phone ports, of which Home Assistant currently support using only one of them.
Also, the Grandstream HT812 has been reported to work. Home Assistant supports using only one of the two phone ports.
If you are running Home Assistant on a machine with other VoIP software, you can configure the port the VoIP integration Integrations connect and integrate Home Assistant with your devices, services, and more. [Learn more] listens on in the configuration. For outgoing calls, you can specify a SIP user value that will be sent to the phone in the From header, if required.
Connecting a phone to Home Assistant requires an adapter.
To add the Voice over IP hub to your Home Assistant instance, use this My button:
Manual configuration steps
If the above My button doesn’t work, you can also perform the following steps
manually:
Browse to your Home Assistant instance.
Go to Settings > Devices & services .
In the bottom right corner, select the
Add Integration button.
From the list, select Voice over IP .
Follow the instructions on screen to complete the setup.
