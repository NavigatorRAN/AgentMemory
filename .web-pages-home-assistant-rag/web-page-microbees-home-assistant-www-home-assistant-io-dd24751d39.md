Configuration
The microBees integration Integrations connect and integrate Home Assistant with your devices, services, and more. [Learn more] allows you to control your microBees devices such as plugs and wall switches.
To use this integration you need OAuth2 Client ID and Client Secret and your user credentials.
To retrieve the OAuth2 Client ID and Client Secret go to microBees Developer Dashboard , log in with your microBees account and create a new app , choose a Label for your App, select WebApplication and input https://my.home-assistant.io as Website URL.
There is currently support for the following device types within Home Assistant:
Switch
Climate
Cover
Binary sensors
Button
Light
Sensor
Note: The cover status will be unknown
To add the microBees hub to your Home Assistant instance, use this My button:
Manual configuration steps
If the above My button doesn’t work, you can also perform the following steps
manually:
Browse to your Home Assistant instance.
Go to Settings > Devices & services .
In the bottom right corner, select the
Add Integration button.
From the list, select microBees .
Follow the instructions on screen to complete the setup.
