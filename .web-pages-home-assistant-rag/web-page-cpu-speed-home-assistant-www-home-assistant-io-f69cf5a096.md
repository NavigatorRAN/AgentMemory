The CPU Speed integration Integrations connect and integrate Home Assistant with your devices, services, and more. [Learn more] allows you to monitor the current CPU speed.
To add the CPU Speed device to your Home Assistant instance, use this My button:
Manual configuration steps
If the above My button doesn’t work, you can also perform the following steps
manually:
Browse to your Home Assistant instance.
Go to Settings > Devices & services .
In the bottom right corner, select the
Add Integration button.
From the list, select CPU Speed .
Follow the instructions on screen to complete the setup.
Not all CPUs are supported. For example, some ARM CPUs
are known not to work with this integration.
