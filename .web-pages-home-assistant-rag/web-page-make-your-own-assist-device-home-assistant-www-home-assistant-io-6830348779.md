If you wish to develop your own Assist device here are some useful advices we can think of:
Make sure you understand the setup and implications of voice Assistants. For reference, we have and we highly recommend you check the information in the developers Portal .
Make sure your setup is going to work with enough quality when adding a voice assistant.
Check hardware options, some options we cover in the documentation are:
ATOM Echo
ESP32-S3-BOX devices
For landline setups be sure to have VOIP
Be sure you check all the addons implied in text-to-speech and speech-to-text, like the Wyoming protocol .
