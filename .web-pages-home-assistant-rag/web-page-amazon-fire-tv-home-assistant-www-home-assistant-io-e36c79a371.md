Support for devices by Amazon Fire TV in Home Assistant is provided by the
Android Debug Bridge
integration.

To add the Amazon Fire TV integration to your Home Assistant instance, use this My button:
Manual configuration steps
If the above My button doesn’t work, you can also perform the following steps
manually:
Browse to your Home Assistant instance.
Go to Settings > Devices & services .
In the bottom right corner, select the
Add Integration button.
From the list, select Amazon Fire TV .
Follow the instructions on screen to complete the setup.
For more documentation on how to use Amazon Fire TV in Home Assistant, please refer to the
Android Debug Bridge integration documentation page .
