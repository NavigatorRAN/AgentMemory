The Roth Touchline integration Integrations connect and integrate Home Assistant with your devices, services, and more. [Learn more] lets you control ROTH Touchline floor heating thermostats from Roth.
To add the Roth Touchline hub to your Home Assistant instance, use this My button:
Manual configuration steps
If the above My button doesn’t work, you can also perform the following steps
manually:
Browse to your Home Assistant instance.
Go to Settings > Devices & services .
In the bottom right corner, select the
Add Integration button.
From the list, select Roth Touchline .
Follow the instructions on screen to complete the setup.
host

The integration will present each Roth Touchline thermostat as a climate entity, which can:
Display the current temperature
Set a target temperature
Assign to a configured Global Schedule using Home Assistant climate entity presets.
