KT-100
KT-200
To add the Kegtron device to your Home Assistant instance, use this My button:
Manual configuration steps
If the above My button doesn’t work, you can also perform the following steps
manually:
Browse to your Home Assistant instance.
Go to Settings > Devices & services .
In the bottom right corner, select the
Add Integration button.
From the list, select Kegtron .
Follow the instructions on screen to complete the setup.
The Kegtron integration will automatically discover devices once the Bluetooth integration is enabled and functional.
The integration only supports Smart Keg Monitor devices of the 1st generation, which communicate via Bluetooth LE.
