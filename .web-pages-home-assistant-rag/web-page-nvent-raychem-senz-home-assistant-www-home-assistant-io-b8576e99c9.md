The nVent RAYCHEM SENZ integration Integrations connect and integrate Home Assistant with your devices, services, and more. [Learn more] allows you to control and monitor your nVent RAYCHEM SENZ-WIFI thermostats.

Manual configuration steps
If the above My button doesn’t work, you can also perform the following steps
manually:
Browse to your Home Assistant instance.
Go to Settings > Devices & services .
In the bottom right corner, select the
Add Integration button.
From the list, select nVent RAYCHEM SENZ .
Follow the instructions on screen to complete the setup.
Control and monitor the thermostat.
Monitor the current temperature.
This integration follows standard integration removal. If you have entered your own credentials, you will be asked if you want to keep them or delete them. If you want to delete them later you can do that from the tree-dot menu in Settings > Devices & services .
Go to Settings > Devices & services and select the integration card.
From the list of devices, select the integration instance you want to remove.
Next to the entry, select the three dots menu. Then, select Delete .
