Manual configuration steps
If the above My button doesn’t work, you can also perform the following steps
manually:
Browse to your Home Assistant instance.
Go to Settings > Devices & services .
In the bottom right corner, select the
Add Integration button.
From the list, select Fujitsu anywAIR .
Follow the instructions on screen to complete the setup.
For more documentation on how to use Fujitsu anywAIR in Home Assistant, please refer to the
Advantage Air integration documentation page .
Fujitsu anywAiR uses port 10211 for API communication. You will need to change the default port provided by the Advantage Air integration to 10211 to use it with Fujitsu anywAiR.
