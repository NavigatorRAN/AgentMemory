Freedompro , a company specialized in home automation, designs and manufactures products to make domotics affordable for everyone, installers and enthusiasts. Freedompro Products are designed to be easy to use and practical and fast to install.
Freedompro with its products installed on the electrical system, allows you to control all the KNX and MyHome SCS devices already present in your home.
This integration lets you control all Freedompro accessories.
You will need to obtain an API key from Freedompro’s app to use this integration.
Download the app on your phone: ( Android / iOS ).
Register and create your first home.
Make sure you have at least one accessory paired with the Freedompro Cloud before starting integration.
Switch to the Profile tab (bottom right corner).
Scroll down to the Developer tools and tap Api Key .
Generate an API key.
To add the Freedompro hub to your Home Assistant instance, use this My button:
Manual configuration steps
If the above My button doesn’t work, you can also perform the following steps
manually:
Browse to your Home Assistant instance.
Go to Settings > Devices & services .
In the bottom right corner, select the
Add Integration button.
From the list, select Freedompro .
Follow the instructions on screen to complete the setup.
