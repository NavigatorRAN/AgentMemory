The Tile integration Integrations connect and integrate Home Assistant with your devices, services, and more. [Learn more] allows Home Assistant to use Tile® Bluetooth trackers .
The official Tile mobile app handles the actual tracking of Tile devices using
the mobile device’s Bluetooth and GPS.

Manual configuration steps
If the above My button doesn’t work, you can also perform the following steps
manually:
Browse to your Home Assistant instance.
Go to Settings > Devices & services .
In the bottom right corner, select the
Add Integration button.
From the list, select Tile .
Follow the instructions on screen to complete the setup.
