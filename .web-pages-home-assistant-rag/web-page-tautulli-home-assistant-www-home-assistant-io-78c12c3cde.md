The Tautulli integration Integrations connect and integrate Home Assistant with your devices, services, and more. [Learn more] will monitor activity on a given Tautulli Server . It will create a sensor that shows the number of currently active streams as the state. If you click the sensor for more details, it will show you more statistics.
To find your api_key , open the Tautulli web interface and navigate to Settings and then to Web interface . The api_key will be at the bottom of that page.
To add the Tautulli hub to your Home Assistant instance, use this My button:
Manual configuration steps
If the above My button doesn’t work, you can also perform the following steps
manually:
Browse to your Home Assistant instance.
Go to Settings > Devices & services .
In the bottom right corner, select the
Add Integration button.
From the list, select Tautulli .
Follow the instructions on screen to complete the setup.
The following stats are available by default:
LAN bandwidth
Number of direct plays
Number of direct streams
Stream count
Top Movie
Top TV Show
Top User
Total bandwidth
Transcode count
WAN bandwidth
