The AirTouch 5 integration Integrations connect and integrate Home Assistant with your devices, services, and more. [Learn more] allows you to control ducted air conditioning systems that are using the AirTouch 5 controller.
To add the AirTouch 5 hub to your Home Assistant instance, use this My button:
Manual configuration steps
If the above My button doesn’t work, you can also perform the following steps
manually:
Browse to your Home Assistant instance.
Go to Settings > Devices & services .
In the bottom right corner, select the
Add Integration button.
From the list, select AirTouch 5 .
Follow the instructions on screen to complete the setup.
The integration will create a climate entity for each AC unit and each zone.
Each zone has a cover entity to represent the state of the damper.
Note
Manual damper adjustments are not recommended in a zone with a temperature sensor, as the AirTouch 5 will automatically control the damper.
