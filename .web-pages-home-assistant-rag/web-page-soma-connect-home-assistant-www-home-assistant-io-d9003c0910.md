Configuration
The Soma Connect integration Integrations connect and integrate Home Assistant with your devices, services, and more. [Learn more] will allow users to integrate their Soma Smarthome devices into Home Assistant using the Soma Connect hub.
You can build a Soma Connect yourself if you have a spare Raspberry Pi. You just need to follow the official instructions . After you have the SD card plug in the Pi and use an ethernet cable or set up Wi-Fi . Then find the IP address by checking your routers DHCP table (we will work on this step).
To add the Soma Connect hub to your Home Assistant instance, use this My button:
Manual configuration steps
If the above My button doesn’t work, you can also perform the following steps
manually:
Browse to your Home Assistant instance.
Go to Settings > Devices & services .
In the bottom right corner, select the
Add Integration button.
From the list, select Soma Connect .
Follow the instructions on screen to complete the setup.
