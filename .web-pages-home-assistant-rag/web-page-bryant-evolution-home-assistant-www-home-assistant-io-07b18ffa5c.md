Configuration
The Bryant Evolution integration Integrations connect and integrate Home Assistant with your devices, services, and more. [Learn more] lets you control Bryant Evolution HVAC systems.
It requires that a System Access Module be connected to the HVAC system (for example, a Bryant SYSTXBBRCT01).
To install:
Connect the System Access Module to the device running Home Assistant, for example, by using an RS-232-to-USB adapter.
Install the Bryant Evolution integration in Home Assistant, passing the name of the serial port to which the device is connected (for example, /dev/ttyUSB0 )
To add the Bryant Evolution device to your Home Assistant instance, use this My button:
Manual configuration steps
If the above My button doesn’t work, you can also perform the following steps
manually:
Browse to your Home Assistant instance.
Go to Settings > Devices & services .
In the bottom right corner, select the
Add Integration button.
From the list, select Bryant Evolution .
Follow the instructions on screen to complete the setup.
