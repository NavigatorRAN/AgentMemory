The PJLink integration Integrations connect and integrate Home Assistant with your devices, services, and more. [Learn more] allows you to control PJLink-compatible projectors from Home Assistant using the PJLink protocol .
To add the PJLink device to your Home Assistant instance, use this My button:
Manual configuration steps
If the above My button doesn’t work, you can also perform the following steps
manually:
Browse to your Home Assistant instance.
Go to Settings > Devices & services .
In the bottom right corner, select the
Add Integration button.
From the list, select PJLink .
Follow the instructions on screen to complete the setup.
Host
