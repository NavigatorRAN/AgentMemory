FTLDNS
powered by Pi-hole®
FTL DNS ™, or simply FTL ( pihole-FTL ), offers DNS blazing fast services within the Pi-hole® project.
It provides DHCP and Web server services. It can also provide TFTP and more as the resolver part based on the popular dnsmasq . Use pihole-FTL -h to learn more about all FTL commands.
Furthermore, FTL offers an interactive API where extensive network analysis data and statistics may be queried.
