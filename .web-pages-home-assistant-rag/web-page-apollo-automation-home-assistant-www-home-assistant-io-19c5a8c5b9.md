Apollo automation is a member of the Made for ESPHome program.
Apollo Automation devices work locally and integrate seamlessly with the ESPHome integration Integrations connect and integrate Home Assistant with your devices, services, and more. [Learn more] in Home Assistant. As all connectivity is done locally, status updates and device control from Home Assistant happen instantly.
Support for devices by Apollo Automation in Home Assistant is provided by the
ESPHome
integration.
Apollo Automation devices are either rebranded devices or devices that share a common
communication protocol, making it possible to use them with the
To add the Apollo Automation integration to your Home Assistant instance, use this My button:
Apollo Automation can be auto-discovered by Home Assistant. If an instance was found,
it will be shown as Discovered . You can then set it up right away.
Manual configuration steps
If it wasn’t discovered automatically, don’t worry! You can set up a
manual integration entry:
Browse to your Home Assistant instance.
Go to Settings > Devices & services .
In the bottom right corner, select the
Add Integration button.
From the list, select Apollo Automation .
Follow the instructions on screen to complete the setup.
For more documentation on how to use Apollo Automation in Home Assistant, please refer to the
ESPHome integration documentation page .
The following devices are known to be supported by the integration. They are certified under the Works with Home Assistant program.
MTR-1 Multi-Target Radar Multisensor (LD2450) - A tiny, but powerful radar-based (mmWave) presence sensor that can pack in many other sensors.
MSR-2 mmWave Multisensor (LD2410B) - An even smaller multisensor that is the successor of the initial community feedback-driven design.
AIR-1 Air Quality Sensor - A small air quality multisensor that can be extended to incorporate a huge variety of sensors.
LED-1 Controller - A versatile LED controller designed for advanced lighting projects.
BTN-1 Macro Deck - A four keycap button deck with LEDs.
R PRO-1 PoE dual mmWave Multisensor - An advanced presence sensor.
PLT-1 / PLT-1B Plant Sensor - An indoor plant multisensor.
