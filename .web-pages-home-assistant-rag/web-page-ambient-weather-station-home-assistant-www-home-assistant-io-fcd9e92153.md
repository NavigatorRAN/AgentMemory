The Ambient Weather Station integration Integrations connect and integrate Home Assistant with your devices, services, and more. [Learn more] retrieves local weather information
via personal weather stations from Ambient Weather .
Using this integration Integrations connect and integrate Home Assistant with your devices, services, and more. [Learn more] requires both an Application Key and an API Key. To
generate both, simply use the profile section of
your Ambient Weather dashboard .
To add the Ambient Weather Station hub to your Home Assistant instance, use this My button:
Manual configuration steps
If the above My button doesn’t work, you can also perform the following steps
manually:
Browse to your Home Assistant instance.
Go to Settings > Devices & services .
In the bottom right corner, select the
Add Integration button.
From the list, select Ambient Weather Station .
Follow the instructions on screen to complete the setup.
This integration communicates with Ambient Weather PWS units via the Ambient Weather
Cloud. Users desiring a local option can use
ecowitt2mqtt
in ambient_weather mode, (which supports MQTT Discovery ).
Another option is to explore the Ecowitt family of devices,
which support the built-in Ecowitt integration.
