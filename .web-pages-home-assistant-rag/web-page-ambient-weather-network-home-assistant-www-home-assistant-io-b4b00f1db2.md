The Ambient Weather Network integration Integrations connect and integrate Home Assistant with your devices, services, and more. [Learn more] retrieves local weather station information
via the Ambient Weather Network .
Similar to the Ambient Weather Station
integration, this integration gathers sensor data from individual weather stations.
However, in contrast to the Ambient Weather Station
integration, which exclusively enables owners to fetch data solely from their owned stations, this
integration directly pulls public data from the Ambient Weather Network without requiring an API key,
albeit with a reduced dataset (for example, excluding indoor sensors).
To add the Ambient Weather Network service to your Home Assistant instance, use this My button:
Manual configuration steps
If the above My button doesn’t work, you can also perform the following steps
manually:
Browse to your Home Assistant instance.
Go to Settings > Devices & services .
In the bottom right corner, select the
Add Integration button.
From the list, select Ambient Weather Network .
Follow the instructions on screen to complete the setup.
