Manual configuration steps
If the above My button doesn’t work, you can also perform the following steps
manually:
Browse to your Home Assistant instance.
Go to Settings > Devices & services .
In the bottom right corner, select the
Add Integration button.
From the list, select RAPT Bluetooth .
Follow the instructions on screen to complete the setup.
The RAPT Pill BLE integration will automatically discover devices once the Bluetooth integration is enabled and functional. RAPT Pill must be switched to the Bluetooth mode in Settings - Telemetry method. See the user manual for detailed instructions.
