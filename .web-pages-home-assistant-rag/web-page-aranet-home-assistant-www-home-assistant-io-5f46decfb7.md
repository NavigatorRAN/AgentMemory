Integrates Aranet devices into Home Assistant.
To add the Aranet device to your Home Assistant instance, use this My button:
Manual configuration steps
If the above My button doesn’t work, you can also perform the following steps
manually:
Browse to your Home Assistant instance.
Go to Settings > Devices & services .
In the bottom right corner, select the
Add Integration button.
From the list, select Aranet .
Follow the instructions on screen to complete the setup.
The Aranet integration will automatically discover devices once the Bluetooth integration is enabled and functional.

The Aranet integration requires that your Aranet device is updated to at least firmware version 1.2.0 and has the “Smart Home integration” feature enabled. Both of these can be done within the settings portion of the Aranet Home mobile application on both Android and iOS.
