The Airly integration Integrations connect and integrate Home Assistant with your devices, services, and more. [Learn more] uses the Airly web service as a source for air quality data for your location.
To generate an Airly API key, go to Airly for developers page.
To add the Airly service to your Home Assistant instance, use this My button:
Manual configuration steps
If the above My button doesn’t work, you can also perform the following steps
manually:
Browse to your Home Assistant instance.
Go to Settings > Devices & services .
In the bottom right corner, select the
Add Integration button.
From the list, select Airly .
Follow the instructions on screen to complete the setup.
Note
Airly allows 100 data updates per day. For this reason, the more Airly instances
configured, the less frequent updates will be. For one configured Airly instance,
data will be updated every 15 minutes, for two configured instances, data will
be updated every 30 minutes, for three configured instances, data will be
updated every 45 minutes, and so on.
