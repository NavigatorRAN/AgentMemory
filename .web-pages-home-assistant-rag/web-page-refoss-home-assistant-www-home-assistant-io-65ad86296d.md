Integrate Refoss devices into Home Assistant.
Devices need to be connected to the local network first.
Devices and Home Assistant must be connected to the same network.
The integration will occupy port: 9989.
To add the Refoss hub to your Home Assistant instance, use this My button:
Manual configuration steps
If the above My button doesn’t work, you can also perform the following steps
manually:
Browse to your Home Assistant instance.
Go to Settings > Devices & services .
In the bottom right corner, select the
Add Integration button.
From the list, select Refoss .
Follow the instructions on screen to complete the setup.
Model Version Refoss Smart Wi-Fi Switch, R10 all Refoss Smart Wi-Fi Switch, R20 all Refoss Smart Energy Monitor, EM06 v2.3.8 and above Refoss Smart Energy Monitor, EM16 v3.1.7 and above
After integrating Refoss, it will start socket broadcast and Home Assistant will automatically discover Refoss devices under the current LAN.
The integration uses the following strategy to name its entities:
The entity name will be generated by Device Name and Channel Number .
Single channel device.
Examples:
Device Name Channel Number Entity Name r10 0 r10
Multi-channel device.
Device Name Channel Number Entity Name r10 0 r10 r10 1 r10-1
Long-press the button on the device to reset it.
If the Refoss device isn’t discovered by Home Assistant automatically, follow these steps:
Verify that your Refoss device and Home Assistant are on the same network
Enable debug logging for the integration:
logger : logs : homeassistant.components.refoss : debug
Restart Home Assistant and attempt to add the integration again
Check the logs for messages containing “Discovered devices”
If no devices are found, verify:
Network connectivity
Device firmware version matches the supported versions listed above
